# Portfolio Frontend

A dynamic, responsive frontend for the professional portfolio website, built with React and Vite.

## Overview

This project serves as the main user-facing portfolio website. It interacts with the Micronaut backend API to fetch and display projects, certificates, resume data, and handle contact form submissions.

## Tech Stack

- **React 19**
- **Vite 8**
- **Tailwind CSS 4**
- **Framer Motion** (Animations)
- **React Router DOM** (Routing)

## Running Locally

1. Clone the repository and navigate to this directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
